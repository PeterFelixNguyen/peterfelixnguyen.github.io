Forest Fire Simulator

This is 2D top-down simulation of a Forest Fire. It is a real-time, interactive simulation with replay fetures and a map editor.

Requirements

Java Virtual Machine, v7 minimum

Links

GitHub repository
Download this app

Features

  • Random map generator: set map size and tree density
  • Map editor: insert and delete trees
  • Replay system: rewind events and change outcomes
  • Adjustable simulation speeds: slow down, speed up, or pause
  • Real-time mechanics: simulation is run in real-time
  • Weather interactions: rain and wind effects

Summary

StatLogic

This is a basic statistical calculator for perform calculations for t-tests, z-tests, confidence intervals, descriptive statistics, and standarad scores.

Requirements

Java Virtual Machine, v7 minimum

Links

GitHub repository
Download this app

Features

  • Hypothesis tests: 1-sample Z-tests and t-tests
  • Confidence intervals: for both t and normal distributions
  • Standard score: quick way to calculate Z-score
  • Descriptive statistics: from a data set, produce descriptive statistics such as mean, median, mode, quartiles, min, max, variance, standard deviation
  • Data sort: sorts values
  • Save and Load: load data files to perform calculations, and save output into a file
