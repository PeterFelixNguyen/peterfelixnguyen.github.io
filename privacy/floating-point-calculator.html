Floating Point Calculator

Privacy Policy

WHAT Data This App Uses

This app uses Android Shared Preferences to store user settings and historical values. Historical values are stored so you can close the app and return to your last session without the value resetting back to "0" in the calculator.

HOW This App Stores Your Data

Android Shared Preferences are stored locally on your Android file system (specifically the Shared Preferences folder in the Floating Point Calculator app). If you uninstall the app, all your selected settings and saved values in the calculator will be cleared.

HOW This App Uses Your Data

All Shared Preferences data are accessed only by the app to load the last session and customize the app functionality.

WHO This App Shares Your Data With

This app does not share your Shared Preferences data.

No one is given access to your data

App Permissions

This app does not use any app permissions.
